Vue Js demo

Valitsin harjoitustyö aiheeksi tutustua Vue Js:n toimintaan tekemällä sillä pienen demo sovelluksen.

Työn nimi: Weather App

Kuvaus: Web-ohjelmoinnin harjoitustyö jonka tein Vue Js:lla demo sovelluksen.

Tekijä: Ville Puisto

Päivämäärä: 13.8.2021

Tein demo sovelluksen jolla pystyy selvittämään millainen sää eri kaupungeissa on sillä hetkellä ja sovellus kertoo päivämäärän, valtion, lämpötilan, millainen ilma on ja taustakuva muuttuu kuvastamaan läpötilaa. Säätilat haetaan käyttämällä API:ta joka saadan Openweathermap:sta se löytää todella monta kaupunkia tai en ainakaan itse huomannut siinä puutteita. Sovelluksen haku kenttään kirjoitetaan kaupunki suomeksi tai englanniksi myös valtion se tunnistaa jonka jälkeen se näyttää säätiedot kaupungista uudella rivillä.

Ajankäyttö:

  • Opiskelu: 14 tuntia
  • Itse ohjelmointi: 11 tuntia
  • Dokumentointi: 5 tuntia

Asennus:

Vuen voi asentaa monella tapaa, itse käytin npm:m ja Vue CLI:tä komennolla npm i -g @vue/cli @vue/cli-service-global ja vue create "nimi" joka vastaa Reactin create-react-app toimintoa. Sitten cd "nimi" kansioon ja sovelluksen "pyörimään" saa komennolla npm run serve.

Itsearvio: Arvosana 3

En tiennyt Vue Js:sta muuta kuin nimen kun rupesin tätä tekemään joten jouduin käyttämään aika paljon aikaa siihen tutustumiseen. Senkin takia ajattelin tehdä sovelluksen jossa ei ole kauhean paljon toiminnallisuuksia jotta sovellus pysyisi yksinkertaisena. Arvosanaksi itselleni ajattelin 3 koska kauhean paljon sovelluksessa ei ole toiminnallisuuksia mutta sain kuitenkin tehtyä Vue:lla sovelluksen joka toimii ja nyt tiedän Vue:sta enemmän ja voisin tehdä isompia sovelluksia joissa olisi enemmän toimintoja kun olen nyt tutustunut siihen.
